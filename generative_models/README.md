# Generative Models

This folder contains implementations of generative learning algorithms.

## Methods

- **VAE (Variational Autoencoder)**: Probabilistic generative model using variational inference
- **Normalizing Flow**: Invertible neural network for density estimation
- **Diffusion Models**: Generative models based on denoising diffusion probabilistic models

## Structure

- `vae/`: VAE implementations and examples
- `normalizing_flow/`: Normalizing flow implementations and examples
- `diffusion/`: Diffusion model implementations and examples
