# Dimensionality Reduction

This folder contains implementations of various dimensionality reduction techniques.

## Methods

- **PCA (Principal Component Analysis)**: Linear dimensionality reduction using orthogonal transformation
- **ICA (Independent Component Analysis)**: Separates multivariate signal into additive subcomponents
- **Autoencoder**: Neural network-based dimensionality reduction

## Structure

- `pca/`: PCA implementations and examples
- `ica/`: ICA implementations and examples
- `autoencoder/`: Autoencoder implementations and examples
